Confidence vs Open-Mindedness

Each agent considers N candidate directions per step (its "rays"), scores each with noise, picks the best, and steps. More rays = more accurate decisions but more time spent thinking. Open-minded usually wins 1v1; a confident swarm usually wins via best-of-many.
